how to export an ableton live session as text

overview

overview

i made a script that allows you to export the names of midi clips in an ableton project recursively to a text file. if you know what youre doing, here:

curl -fsSL https://mote.moe/scripts/ableton_export_text.sh | cat

or if you trust me and youre very brave:

curl -fsSL https://mote.moe/scripts/ableton_export_text.sh | bash

otherwise, lets go through everything the script does in this post.
